# Phase 4 — استيراد بيانات المدارس (Excel)

**الهدف:** رفع Excel → معاينة → كشف تكرار → تحديث ذكي → سجل عمليات.

**يعتمد على:** Phase 1 (مدارس/أقسام/تصنيفات) + Phase 2 (السياق).

---

## المهام

### قاعدة البيانات
- [ ] Migrations: `coordinators`, `teachers`, `import_batches`, `import_errors`. (انظر [`02-DATABASE.md`](../02-DATABASE.md))

### المنطق
- [ ] تثبيت `maatwebsite/excel` (Laravel Excel).
- [ ] `ImportSchoolDataJob` (مؤجّل للملفات الكبيرة):
  - أعمدة Excel: المدرسة، المرحلة، المادة، المنسق، المعلم، التصنيف، عدد الشعب (IM-1).
  - **معاينة قبل الحفظ** (parse → عرض → تأكيد) (IM-2).
  - **كشف التكرار + تحديث ذكي** (upsert حسب مفاتيح طبيعية) (IM-3).
  - تسجيل الأخطاء في `import_errors` + ملخّص في `import_batches` (IM-4).
- [ ] `ImportService` + DTO لكل صف + Validation.
- [ ] قالب Excel للتنزيل (template).

### الواجهة
- [ ] شاشة الاستيراد: **React Dropzone** لرفع الملف.
- [ ] جدول معاينة (صحيح/مكرّر/خطأ) قبل التأكيد.
- [ ] شريط تقدّم + نتيجة (مستورد/محدّث/فاشل).
- [ ] **سجل عمليات الاستيراد** + عرض الأخطاء + إعادة محاولة.
- [ ] الموجه يستورد لمدارسه فقط ضمن العام الفعّال (IM-5).

---

## معايير القبول
- رفع ملف صحيح → معاينة → حفظ ينشئ/يحدّث المدارس والمنسقين والمعلمين.
- التكرار يُكتشف ويُحدّث بدل التكرار.
- الأخطاء تُعرض بوضوح وقابلة للمراجعة.
- كل البيانات مرتبطة بالعام الفعّال.
