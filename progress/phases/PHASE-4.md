## ما تم فعليًا
- **الحزمة:** `maatwebsite/excel` 3.1 + phpspreadsheet.
- **الخدمة:** `ImportService` — `parse()` (تحليل xlsx/xls/csv + ربط الأعمدة العربية بمرونة)، `preview()` (حالة كل صف: جديد/تحديث/خطأ + ملخّص)، `import()` (تحديث ذكي عبر `updateOrCreate` للمدارس/المنسقين/المعلمين + تسجيل الدفعة والأخطاء + ملخّص).
- **التدفق:** رفع → تخزين مؤقت → معاينة (token) → تأكيد → استيراد → حذف المؤقت. الاستيراد مقيَّد بالعام النشط (`isEditable`).
- **الواجهة:** `import/index` — Dropzone، StatCards للمعاينة (جديد/تحديث/أخطاء)، جدول معاينة، سجل الدفعات، عرض الأخطاء، تنزيل القالب (`SchoolTemplateExport`).
- **اختبارات:** `ImportTest` (6): التحليل، المعاينة، إنشاء الكيانات + الدفعة، **التحديث بدل التكرار**، الصفحة، الصلاحية. **إجمالي المشروع: 65 اختبار / 229 توكيد يمر.**

## قواعد العمل المطبّقة
IM-1 (الأعمدة السبعة)، IM-2 (معاينة قبل الحفظ)، IM-3 (كشف تكرار + تحديث ذكي)، IM-4 (سجل دفعات + أخطاء)، IM-5 (مرتبط بالعام النشط).
