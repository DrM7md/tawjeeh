## ما تم فعليًا
- **الجداول:** `visits` (morph على teacher/coordinator، مرتبط بالعام **والفصل**) + `visit_forms` (1:1، snapshot + axes + draft/final) + `visit_files` (تخزين خاص).
- **المنطق:** `VisitService` — record، followUp (أهداف ضمن النطاق + حالة الإنجاز: required من التصنيف للمعلم و1 للمنسق، done سنويًا عبر كل الفصول، حالة تمت/متبقٍ/متأخر) + قائمة الزيارات مع `scopeVisibleTo`.
- **الاستمارة:** `VisitFormService` — مسودة/اعتماد نهائي (قفل بعد الاعتماد إلا بصلاحية `forms.finalize`) + مرفقات في `storage/app/private` (disk local) + تنزيل عبر مسار محمي.
- **الواجهة:** `visits/index` (StatCards إنجاز/تمت/متبقٍ/متأخر + لوحة متابعة بزر «تسجيل زيارة» + قائمة الزيارات + فلتر القسم) و`visits/show` (محاور تقييم + ملاحظات/توصيات + توقيع + مرفقات + حفظ مسودة/اعتماد، وضع قراءة فقط بعد الاعتماد).
- **اختبارات:** `VisitTest` (8): السياق، تدرّج الحالة، المنسق، مسودة/اعتماد، نطاق التسجيل، رؤية الموجه، رفع ملف خاص، الصفحة. **إجمالي المشروع: 73 اختبار / 261 توكيد يمر.**
